All Components

All components included with the Ardisia Component Library are designed to be used with the Flex SDK.

Accordion
Custom viewstack component.
AnimatedImage
Bitmap animations with granular control over framerates.
AutoComplete
Real-time DropDownList filtering.
AutoFitText
Text components that automatically size to fit the available space.
BarCode
Generates and displays various bar codes.
BitmapAdjustments
Effects and changes for bitmap data.
ButtonBar
Button bar that supports dragging, closing, and animation.
Calculator
Stand-alone or grid editor calculator.
Calendar
Full featured Calendar.
CAPTCHA
Audio and/or visual test if the user is human.
CarouselLayout
CarouselLayout generalized layout class.
CarouselList
List optimized for use by the CarouselLayout class.
Clocks
Analog and digital clock.
Colorizer
Component that returns colors based on passed values.
ColorPicker
Spark color picker with enhanced customization.
ColorSpacePicker
Color space picker.
CompassContainer
Container with north, east, south, west, and center regions.
CoverflowLayout
Coverflow generalized layout class.
CoverflowList
List optimized for use by the CoverflowLayout class.
CursorManager
Add, display, and manage native mouse cursors.
DatePicker
Spark based DatePicker that supports skinning and custom renderers.
DockingCompassContainer
CompassContainer that allows dragging and docking of regions.
Donut Chart
Supports several rings of data series.
ExpandingContainer
Expandable/collapsible skinnable container.
FieldSet
Fieldset container similiar to a HTML fieldset.
Filters
Additional bitmap filters.
FlexContextMenu
Context menu that opens on right-click.
FloatPaneControlBar
Control bar used to manage floated Panes.
Funnel Chart
Chart that displays data in a funnel pattern.
GraphicsEditor
Graphics editor.
Heatmap
Chart that displays data via colorizers.
HtmlDragDrop
Accept file drops from the user's desktop via the HTML 5 API.
HtmlFrame
Display html content inside a flex application.
HtmlGeolocation
Retrieve user location via HTML 5 API.
iCalendarParser
Exports/imports iCalendar formatted strings.
iCalendarRecurrence
Generates and returns recurring dates based on the iCalendar spec.
IconButton
A button designed to display icons with a flyout label.
Linear Gauges
Horizontal and vertical gauge sliders with advanced features.
Magnifier
Apply a magnification to visual components.
Menu and MenuBar
Extended Halo Menu and MenuBar that adds customization options.
MiniViewport
Small viewport designed to help the user navigate scrollable content.
PackedLayout
Layout that minimizes element overlap.
PackedList
PackedLayout List with support for throwing.
Pane
Containers that can be floated and behave like windows.
PopUpButton
Button that can also open popup content.
ProgressDisplay
Progress indicators.
Radar
Also known as a polar, spider chart, or a kiviat diagram.
Radial Gauge
Circular gauge used to select numerical values.
Rating
Used to rate content.
ReflectionContainer
Simple and performant reflection container.
RichTextEditor
Toolbar that manages Text Layout Framework based controls.
RotaryField
Control used to select angles.
SafariMouseWheelFix
Adds mouse wheel support to content running in Safari.
Scroller
Scroller with granular control over the pixels scrolled.
Separators
Simple analog to a HTML HR display element with padding support.
SliderField
Control used to select a numerical value.
Sparklines
Small bar and line charts useful in grids.
TabPaneNavigator
Tabbed viewstack designed for docking Pane containers.
TextInput
TextInput with icon and clearing support.
TimebarContainer
Container used to select and display time periods.
Timeline
Display events along a Timebar.
TimeMachineLayout
TimeMachine generalized layout class.
TimeMachineList
List optimized for use by the TimeMachineLayout class.
TimePicker
Control used to select a time, specific to the minute.
ToggleSwitchDesktop
ToggleSwitch component designed for desktop use.
Toolbar
Container used to display toolbar content.
TransformContainer
Enables dragging and transformations for child content.
TreeMap
Displays hierarchical data by using nested rectangles.
Utilities
Mountain of utility classes.
ViewStack
Container that displays one child element at a time.
WheelList
A list with renderers arranged as slices of an ellipse.
